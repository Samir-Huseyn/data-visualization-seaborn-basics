# 📊 Supermarket Sales Data Analysis

## 📌 Project Overview
This project analyzes a supermarket sales dataset using Python, Pandas, Matplotlib, and Seaborn.  
It explores customer behavior, product performance, and revenue distribution through data analysis and visualization.

---

## 📂 Dataset
- File: SuperMarket_Analysis.csv
- Contains sales data including:
  - City
  - Gender
  - Product line
  - Quantity
  - Payment method
  - Gross income

---

## 🧠 Key Analysis
The project answers the following questions:

- Which city generates the highest income?
- Which product line sells the most?
- Which gender buys more products?
- What is the most used payment method?
- What is the average total spending?

---

## 📊 Visualizations
The following visualizations are included:

- 📌 Sales by City (Bar Chart)
- 📌 Sales by Product Line (Bar Chart)
- 📌 Gender Distribution (Count Plot)
- 📌 Payment Methods (Count Plot)
- 📌 Income Distribution (Histogram)
- 📌 Income by City (Box Plot)
